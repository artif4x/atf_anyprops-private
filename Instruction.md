# ATF Anyprops

ระบบที่ให้ผู้เล่นสามารถวางไอเทม อาวุธ หรือของตกแต่งต่างๆ ลงบนพื้นหรือบนรถได้อย่างสมจริง โดยระบบจะอ้างอิงไอเทมที่มีอยู่ในกระเป๋าของผู้เล่นจริง

## Dependencies
- qbx_core
- ox_inventory
- ox_lib
- ox_target

## Installation
1. วางสคริปต์ใน resources ของ Server หรือใน [standalone] ก็ได้
2. หากอยู่ใน [standalone] ไม่ต้องเพิ่มอะไร (เช็คว่าใน server.cfg มี ensure [standalone] ไว้หรือยัง ถ้าไม่อยู่ ให้เพิ่มคำสั่ง `ensure atf_anyprops` ลงในไฟล์ server.cfg)
3. ตรวจสอบให้แน่ใจว่าติดตั้ง dependencies ครบถ้วนตามด้านบนแล้ว

## Setup

เปิดไฟล์ `config.lua` เพื่อตั้งค่าเบื้องต้น
- Webhook: ใส่ลิงก์ Discord Webhook เพื่อใช้รับข้อมูลการวางของและการเก็บของ เป็น Discord Log แบบ Queue ทำให้การส่งข้อมูลไม่ทำให้ Server กระตุก
- Roleplay: สามารถตั้งค่าเปิดปิดข้อความแสดงสถานะ (/me) และเลือกรูปแบบคำสั่งได้ 
- Vehicle Crash: ปรับตั้งค่าความเสียหายของรถที่จะทำให้ของที่วางอยู่ร่วงหล่นได้ที่นี่ 
- Vehicle Roll: ปรับแต่งองศารถคว่ำ กลิ้ง แล้วของตกได้
- Items: เพิ่มไอเทมใหม่ที่อยากให้ถือได้ (ต้องไม่ใช่ไอเทมที่กดใช้ได้อยู่แล้ว) โดยเพิ่มใน config.lus และเพิ่มใน ox_inventory/data/items.lua


## How to use
1. การวางของ
กดใช้ไอเทมจาก Inventory ตัวละครจะถือไอเทมชิ้นนั้น จากนั้นกดปุ่ม E เพื่อเริ่มโหมดวางของ 

2. การวางอาวุธ
ถืออาวุธ แล้วกดปุ่ม E (หรือปุ่มที่ตั้งค่าไว้) ขณะถืออาวุธ เพื่อวางอาวุธลงพื้น 

3. โหมดวางของ
* ใช้ปุ่มลูกศร หรือ Scroll เมาส์เพื่อหมุนของ
* ใช้ปุ่ม [ หรือ ] เพื่อเลื่อนของขึ้นหรือลง (ปรับได้ใน Settings > Key Bindings > FiveM)
* กดปุ่ม G เพื่อสลับโหมดติดหนึบกับรถ
* กดปุ่ม L เพื่อพลิกของ 90 องศา (ปรับได้ใน Settings > Key Bindings > FiveM)
* กดปุ่ม ; เพื่อวางของให้ติดพื้น (ปรับได้ใน Settings > Key Bindings > FiveM)
* กดปุ่ม ' เพื่อรีเซ็ตมุมมอง (ปรับได้ใน Settings > Key Bindings > FiveM)
* กดปุ่ม Enter เพื่อวางของให้เสร็จสิ้น หรือ Backspace เพื่อยกเลิก 

4. การเก็บของ
เดินไปใกล้ๆ ไอเทมที่วางอยู่ 
ถ้าเป็น Props ให้กด L Alt เพื่อใช้ ox_target
ถ้าเป็น อาวุธ กดปุ่ม Z (หรือปุ่มที่ตั้งค่าไว้) เพื่อเก็บคืนเข้ากระเป๋า

โดยของที่วาง และของที่เก็บคืน จะเป็นของชิ้นเดียวกัน ไม่มีการรีเซ็ตความทนทาน หรือเสกของใหม่ให้
อาวุธก็จะมี Serial Number, จำนวนกระสุนเท่าเดิม

## Admin Tools

- `/editprop` : เปิดโหมดปรับแต่งพิกัด Prop ที่ถือ ถ้า Prop เบี้ยว แล้วกด Enter เอาค่าจาก F8 ไปใส่ใน config.lua ของไอเทมชิ้นนั้น
 
- `/prop_mass_pickup [รัศมี]` : สั่งกวาดเก็บไอเทมทั้งหมดในรัศมีที่กำหนด เพื่อเคลียร์ขยะ
 
- `/atf_debug` : เปิดโหมดแสดงจุดสีแดงเพื่อดูตำแหน่งของที่วางอยู่จากสคริปต์นี้

## Performance Testing Tools (ใช้เฉพาะตอนเทสเซิฟ)

- `/prop_test_spawn [จำนวน]` : สั่งเสก prop มาบนพื้นรอบๆ ตัว พร้อมกันตามจำนวน (ตั้ง Default ไว้เป็นกรวยจราจร)
